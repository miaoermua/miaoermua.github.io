CatWrt-Newifi D2 新路由3鸡血 MTK 无线固件

2022-08-10- 2022-08-17
706- 2m- -
硬路由-硬路由固件OpenWrt
喵二

CatWrt For Newifi D2 新路由3鸡血 MTK 无线固件，因为市面上的新3的固件都不是太适合我。首先我需要鸡血无线和够用的功能也就是当无线 ap 使用，所以就有了这版固件。

特色功能

  • 大雕源码最小化
  • MTK 鸡血 wifi 无线驱动
  • 完全支持 IPv6
  • 基础网络加速
  • 普通远程管理

因为 Newifi D2 的存储空间过小只有可怜的 32m 所以专门做了个 CatWrt 最小化固件，和 x86 版 v22.7 基本上是一个源码，所以插件除了原版+内网穿透其他全部为打包好的插件有需要可以手动安装

配置

默认IP 192.168.1.4

用户 root

密码 password

你也可以使用默认不修改，可选项；如果你的网络之前是别的网段请按照下方修改。

自动开启wifi

因为鸡血固件特性，只能通过这样的方式打开无线，以下方法都建议使用。

电脑有线设置静态地址为 192.168.1.x 子网掩码 255.255.255.0,主要的网关/路由器地址填 192.168.1.4，DNS留空或者和网关一样

脚本

登录 OpenWrt 后台，在系统-启动项-本地启动脚本 插入 /sbin/mtkwifi up 在 exit 0 前面。

完整脚本

/sbin/mtkwifi up

exit 0

计划任务

在系统-计划任务中填入 0 * * * * /sbin/mtkwifi up 计划任务。

完整计划任务

0 * * * * /sbin/mtkwifi up

修改默认 IP

在 OpenWrt 终端中输入 vi /etc/config/network 在窗口修改 option ipaddr 行中的 IP 地址。

完成后按 Esc，再按下冒号 :，在输入框中输入 wq；实际显示 :wq

返回后输入 reboot 重启。

修改默认密码

在终端里输入 passwd 输入密码，后确认密码。
