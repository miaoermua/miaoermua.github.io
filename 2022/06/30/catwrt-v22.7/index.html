📣 最新版 v22.8 解决了英特尔 i225 2.5G 网卡表现不佳的问题，强烈建议使用 CatWrt v22.8

在去年的10月10日，我公开发布了我编译的 OpenWrt 软路由固件，这次更新了带来了新特性和特色功能，经久耐用的固件主旁单臂都很稳定高效。

基于 Lean 源码 拉取 2022-6-28 的仓库，精心打造！

特色功能

  • 令人惊叹的远程管理
  • 完全支持 IPv6
  • 智能网络加速
  • 负载均衡
  • Docker

配置

默认IP 192.168.1.4

用户 root

密码 password

你也可以使用默认不修改，可选项；如果你的网络之前是别的网段请按照下方修改。

修改默认 IP

在 OpenWrt 终端中输入 vi /etc/config/network 在窗口修改 option ipaddr 行中的 IP 地址。

完成后按 Esc，再按下冒号 :，在输入框中输入 wq；实际显示 :wq

返回后输入 reboot 重启。

修改默认密码

在终端里输入 passwd 输入密码，后确认密码。

更新预览

查看编译时选择的插件预览 获取 （不包含插件依赖）

  • 新增 frps，nezha-agent，argon-config，poweroff，neobird
  • 移除 过时少用定位冲突插件
  • 修复 argon，防火墙
  • 魔改主题 argon， neobird
  • 修改主机名为 CatWrt
  • 修改默认 IP 192.168.1.4
  • 修改 banner 个性易辨别
  • 更改 linux 内核占用 25M
  • 更改 ovenlay 分区占用 300M

文件位置 v22.7/targets/x86/64

squashfs-combined 为传统 BIOS 启动

squashfs-combined-efi 为 UEFI 启动

img 文件建议使用 IMG写盘工具 或者其他写盘工具进行安装

qcow2 文件是 pve 或 kvm 虚拟机的虚拟硬盘文件

vhdx 文件是 hype-v 虚拟机的虚拟硬盘文件

vmdk 文件是 vmware 虚拟机的虚拟硬盘文件
