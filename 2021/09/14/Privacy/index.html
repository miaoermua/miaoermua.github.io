Privacy 隐私

我站非常重视您的个人隐私和企业隐私，我们在2021年9月3日开始实行隐私管理条例；隐私政策 更新于2022年2月6日；并生效于2021年9月14日。

本站以下地址均生效:
https://www.miaoer.xyz，https://blog.miaoer.xyz

我们不会和如何个人或企业共享你的隐私数据，如有疑问请在联系我。

Only in Chinese, if you use the translation software for reference only.

我们是怎么收集你的隐私

  • 访问

当你访问我们提供的站点服务时我们将收集你的访问 IP ，该条例按照工业和信息化要求保留3个月左右但我们进行无限期的保留。

  • 评论

当你使用我们提供的站点评论服务时我们将收集你的访问 IP 和浏览器的 User-Agent，以及您发送的评论图片信息，我们将无限期保留并且用于评论展示。

  • 统计

我们通过浏览器单次访问进行统计阅读量和访问量，在后端或者数据库中用于提高我们服务质量。

  • Cookies

当你访问我们提供的站点服务时我们会在你的浏览器中生成 Cookies，以完成网站基本服务（如: 评论、统计、提高百度SEO值），你可以使用后完全清理我们的Cookies，我们不会利用Cookies收集个人隐私。

  • Google Analytics

我们通过Google Analytics 谷歌分析采集你在本站(如:网页浏览量，滚动次数，出站点击次数，网站搜索，文件下载次数)来提升本站预览服务；Google分析 详情描述。

我们怎么利用你的隐私

我们会不定期通过 Google Analytics 分析用户行为以优化网页用户体验；请放心 Google Analytics 在中国大陆地区可以正常收集，不需要通过 科学上网，如果你在 中国大陆境外 或者 开启代理网络 访问的情况下可能会消耗你的出口数据流量或者漫游数据。

当你访问本站并且产生的相关信息(如: IP 地址，浏览器的 User-Agent，以及您发送的评论图片信息…)会加密保存到我们的后端数据库缓存中，管理员面板此处简称 后端 可以查看，我们可能将无限期进行保留，如当地监管部门需要我们会如实提供，我们不会和境内外非法组织政企 含NGO ，提供任何包含你个人信息的内容。

当你访问个人主页我们会使用后端进行数据管理，你的 IP 地址会通过内置的 IP 查询 API 进行查询；当你访问国际站( github.io )时我们没有权利追踪你的 IP 和隐私。

您对您的产生的信息有什么权利

  • 评论

我们将提供评论撤回功能，并且您有权利将您信息在数据库保留三个月后进行删除操作。

  • Cookies

你可以使用我站后完全清理我们的 Cookies。

本隐私条款会随着时间逐步完善更新，本站会在告示下进行本条例的修改或更新。坚决杜绝霸王条款，坚决捍卫守护用户的隐私行为！
