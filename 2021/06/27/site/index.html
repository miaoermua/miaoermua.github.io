    提交PR 教程

    • 先 Fork miaoermua.github.io

    • 进入 提交PR 点击小笔头 "Edit this file"

    • 在编辑模式下增加你的友链信息

      • 复制一下上面已有的友链内容粘贴到代码最地下替换掉 ] ，当你不想写邮箱请把 ,"email": "邮箱" 字段删除。
      不会可以看着友链上面的抄，怕打错我给你修，不过会提PR的人应该都会基础的；或者看结合代码中的内容再看下面代码块也许你就明白了
      , { "url":"博客链接", "avatar":"博客头像地址", "name":"昵称", "blog":"博客名称", "desc":"个性描述", "color": "颜色", "email": "邮箱" } ]
    • 标题可以写点博客名字无要求，点击 Propose changes 在新页面点击 Pull Request 将你的友链信息并入 PR

    • 等待PR审核
