友链已经足够多了，但我们还继续收入友链，为了保证友链质量我们会筛选友链；

但我们支持个人新人博主良好发展，请使用备案服务器和域名、Https来提高你的友链质量！下架友链可能不另外进行通知。



为什么友链消失或不通过呢？

因为博客含有以下内容，不通过。

如果检查到不通过的友链，我会向你申请的邮箱发送问题原因，假如你修复了，再通过友链申请。

申请失败:

违规内容 (条例按照: 博客内容不应该出现传统意义上的违规内容 合理: 不包含 涉政，涉黄内容)

博客无法访问 (条例按照: 博客在大部分时间都不可访问 合理: 服务器在大部分时间都因处于开启并可访问状态)

博客动件卡顿 (条例按照: 博客比YouTube播放4K视频占用更高 合理: 瞬间加载占用后保持低占用)

访问速度极慢 (条例按照: 在国内互联网环境下访问速度极慢 合理: 就算是代理软件环境下你的博客都不能太慢，至少能访问)

友链关系不稳定 (条例按照: 友链页面包含我站内容不应该出现消失后几天又恢复 合理: 不要删除后再恢复反复横跳，假如你要删除友链可以适当通知我，友链不应该出现这种问题)

提交你的友链

现支持 直接提交 ， 提交PR ， Telegram ， 邮箱 来提交你的友链。

可以先联系我再添加我的友链，如果你单向添加也无所谓

提交PR 教程

  • 先 Fork miaoermua.github.io

  • 进入 提交PR 点击小笔头 "Edit this file"

  • 在编辑模式下增加你的友链信息

    • 复制一下上面已有的友链内容粘贴到代码最地下替换掉 ] ，当你不想写邮箱请把 ,"email": "邮箱" 字段删除。
    不会可以看着友链上面的抄，怕打错我给你修，不过会提PR的人应该都会基础的；或者看结合代码中的内容再看下面代码块也许你就明白了
    , { "url":"博客链接", "avatar":"博客头像地址", "name":"昵称", "blog":"博客名称", "desc":"个性描述", "color": "颜色", "email": "邮箱" } ]
  • 标题可以写点博客名字无要求，点击 Propose changes 在新页面点击 Pull Request 将你的友链信息并入 PR

  • 等待PR审核

  • 等待 cdn 刷新
